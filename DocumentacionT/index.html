Frontend

¿Qué es?

Es la parte visible de una aplicación web o móvil, con la cual el usuario puede interactuar, su enfoque esta centrado en la experiencia e interfaz de usuario.

¿Qué tecnologías se usan?

HTML, CSS, JavaScript, también se pueden utilizar framworks y librerias como React, Angular, Vue.js, etc

Objetivo

Crear una aplicación web o móvil que sea fácil de usar, intuitiva y agradable para el usuario.

Hello Word con JavaScript

console.log("Hello World")
Backend

¿Qué es?

Es la parte de una aplicación que se encarga de la lógica de negocio, el almacenamiento y la gestión de datos, así como la comunicación con el frontend.

¿Qué tecnologías se usan?

Python, Java, Node.js, Ruby, PHP; bases de datos como MySQL, PostgreSQL, MongoDB; y frameworks como Django, Spring, Express.js, entre otros.

Objetivo

Asegurar el gestionamiento de información y seguridad

print("Hello World")
Game Development

Es el proceso de crear videojuegos, que incluye la programación, diseño gráfico, sonido y narrativa.

¿Qué tecnologías se usan?

C#, C++, JavaScript, y lenguajes de scripting específicos de cada motor.

Ciberseguridad

¿Qué es?

Es la protección de sistemas informáticos, redes y datos contra accesos no autorizados, uso no autorizado, revelación, alteración o destrucción.

¿Qué tecnologías se usan?

Firewalls, antivirus, VPN, sistemas de autenticación, etc.

Objetivo

Prevenir y detectar amenazas a la seguridad de la información

Inteligencia Artificial

¿Qué es?

Es la capacidad de un sistema informático de realizar tareas que normalmente requieren la inteligencia humana, como el aprendizaje, la resolución de problemas y la toma de decisiones.

¿Qué tecnologías se usan?

Python, R, Java, C++, y frameworks como TensorFlow, Keras, PyTorch, etc.

Objetivo

Crear sistemas que puedan aprender y mejorar con la experiencia
